16.1.1 Funci√≥n stem ()

Esta funci√≥n permite crear el gr√°fico llamado de tallo y hoja. Este gr√°fico fue propuesto por Tukey (1977) y a pesar de no ser un gr√°fico para presentaci√≥n muy usado comunmente, se puede utilizar a la vez que el investigador recoge la informaci√≥n para ver r√°pidamente la distribuci√≥n de los datos.

¬øQu√© muestra este gr√°fico?

  2. La forma general de la distribuci√≥n:

  2. F√°cil de entender.

Desventajas del gr√°fico:

  1. El gr√°fico es rudimentario y raras veces se usa en presentaciones y/o art√≠culos cient√≠ficos .
  2. Funciona cuando el n√∫mero de observaciones no es muy grande. No permite comparar claramente diferentes poblaciones

16.1.4 Funci√≥n density ()

La funci√≥n density () hace una estimaci√≥n suave de la distribuci√≥n de los datos (el histograma suele ser m√°s abrupto). Para ello usa una estimaci√≥n no param√©trica usando kernels, que es una metodolog√≠a avanzada en estad√≠stica. Una vez estimada la densidad de nuestra variable de inter√©s, basta con usar la funci√≥n plot () para crear el gr√°fico. Los argumentos de la funci√≥n density () son:

function (x, ...) 
NULL

donde:

  • x: es el vector con los datos para los cuales se quiere la densidad.
  • bw: ancho de banda (se usa para suavizar m√°s o menos la gr√°fica).
  • kernel: n√∫cleo de suavizaci√≥n a usar, los posibles valores son gaussian, rectangular, triangular, epanechnikov, biweight, cosine o optcosine, el valor por defecto es gaussian. Esto es muy avanzado y no suele cambiarse.

Ejemplo

Veamos, de nuevo, c√≥mo describir nuesrta variable de inter√©s mediante su densidad. Como hay missings en nuestros datos, debemos decirle que los elimine (NOTA: si no a√±adimos na.rm=TRUE obtendremos un error)

El siguiente c√≥digo mostrar√≠a c√≥mo afecta el ancho de banda en la estimaci√≥n:

Podemos observar como dar un par√°metro de suavizado muy peque√±o (0.2) crea una estimaci√≥n muy ruidosa, mientras que usar un valor muy grande (10) puede llevar a conclusiones err√≥neas ya que con este valor parece que los datos sean normales cuando cl√°ramente estan sesgados hacia la izquierda (edades j√≥venes). Cabe rese√±ar que estos valores de suavizado dependen de la escala que tengamos en nuestra variable de inter√©s.

Tambi√©n podemos obtener la misma distribuci√≥n separada para casos y controles. Para ello necesitamos crear una base de datos distinta para cada grupo (esto es una desventaja de usar funciones b√°sicas de R, pero tampoco es tan grave). Podemos usar las funciones que ya conocemos para tratar datos. Crearemos la densidad para uno de los grupos y luego a√±adiremos la del otro con la funci√≥n lines ().      xlab='Estado (caso/control)', lwd=4, col='blue')

       lwd=3, col=c('blue', 'red'))

Podemos observar como los controles tienen su primer embarazo a edades m√°s tard√≠as.

16.3.1 Funci√≥n barplot ()

Los gr√°ficos de barras son √∫tiles para representar las frecuencias absolutas o relativas asociadas a los niveles de una variable cualitativa. La funci√≥n barplot tiene otros par√°metros que pueden verse ejecutado ?barplot.
